## 1. 项目概述

### 1.1 项目目标

构建企业级 API 测试自动化框架，展示以下能力：
- RESTful API 全面测试
- Postman 集合开发
- Newman CLI 自动化执行
- 环境配置管理
- 测试脚本编写 (JavaScript)
- 测试数据管理

### 1.2 业务场景

电商订单管理系统 (E-commerce Order Management System)
