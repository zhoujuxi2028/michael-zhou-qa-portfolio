# CI / 本地测试日志

> **TL;DR:** 实际日志文件**不入仓库**，统一从 GitHub Actions Artifacts 或本地工作目录获取。

---

## 1. 为何不入仓库

- 单个 CI 日志可达 100KB+（如 `ci-run-23992786381.log` ≈ 152KB），混入文档目录会污染 diff、放大 git 体积。
- 日志属于**易逝过程产物**，与 Stage Gate / RCA 等"长期资产"性质相反。
- `.gitignore` 已规则化覆盖：`docs/qa/reports/logs-*/`、`docs/qa/reports/*.log`。

---

## 2. 如何获取

| 来源                                | 方式                                                                                                                            |
| ----------------------------------- | ------------------------------------------------------------------------------------------------------------------------------- |
| **GitHub Actions Artifacts**        | 进入对应 workflow run → 右上 `Artifacts` 区下载（保留 90 天）                                                                   |
| **`gh` CLI**                        | `gh run view <run-id> --log` 或 `gh run download <run-id>`                                                                      |
| **本地 Gate 自检**                  | `bash scripts/p0-gate-check.sh` / `p1-gate-check.sh` 默认输出到 `docs/qa/reports/logs-p0/` 与 `logs-p1/`（**本地目录，不入仓库**） |
| **本地集成测试**                    | `bash scripts/integration-test.sh` 输出到 `tests/integration/logs/`（同样不入仓库）                                              |

---

## 3. 历史日志归档

| 日期       | 来源                          | 当前位置                           | 说明                              |
| ---------- | ----------------------------- | ---------------------------------- | --------------------------------- |
| 2026-04-05 | CI run 23992786381 / 24001840882 | _(已从仓库移除)_                | 如需追溯，使用 `gh run view 23992786381 --log` |

历史日志若被 RCA / postmortem 引用，引用方应当：

1. 摘录关键片段（10~30 行）直接嵌入 RCA 文档，**不依赖外部日志**；
2. 若必须保留完整日志，附 `gh run view <id>` 命令而非提交日志文件。

---

## 4. 报告与日志的关系

| 资产类型 | 示例                                            | 是否入仓库 |
| -------- | ----------------------------------------------- | ---------- |
| 报告     | `reports/execution/stage4-execution-*.md`       | ✅ 入仓库  |
| 摘录     | RCA 文档中嵌入的 10~30 行日志片段               | ✅ 入仓库  |
| 完整日志 | `*.log`、`logs-<date>/` 整个目录                | ❌ 不入仓库 |
