推荐配置

推荐：情况③ — SSH + SOCKS5 :7891

写入 ~/.ssh/config 后，所有 git push/pull 自动生效，无需额外参数： 
Host github.com
  ProxyCommand nc -x 127.0.0.1:7891 %h %p
  ServerAliveInterval 30
  ServerAliveCountMax 10
验证命令：ssh -T git@github.com

注意：Bywave 需保持运行，SOCKS5 端口 7891 需已开启。偶发超时时可重试，通常第二次即正常。
